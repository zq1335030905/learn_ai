# 知识库 (Knowledge Base)

> 结构化沉淀，可复用的知识资产。

## 核心结构

- **`foundations/`**: Android 图形系统底层原理（EGL / BufferQueue / SurfaceFlinger / Vulkan 等）
- **`patterns/`**: 应用落地的范式与模式（Camera Preview / Video Render / Compositor 等）
- **`concepts/`**: 关键概念辨析（术语、边界、反例、常见误解）

## 使用方式

1. 新概念/机制 → 先定位到概念地图（`foundations/graphics-concept-map.md`）
2. 按模板创建条目（`TEMPLATE.md`）
3. 确保与掌握证据（`learning-log/mastery/`）双向链接
