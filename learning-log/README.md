# 学习日志 (Learning Log)

> 记录学习过程，沉淀可复用的经验。

## 目录结构

```
learning-log/
├── INBOX.md                    # 即时捕捉（会话中随时追加）
├── TEMPLATE.md                 # 每日日志模板
├── WEEKLY_REVIEW_TEMPLATE.md   # 周复盘模板
├── mastery/                    # 掌握证据
│   ├── TEMPLATE.md
│   └── <concept>.md
└── YYYY-MM-DD-<topic>.md       # 每日学习日志
```

## 使用流程

1. **会话中**：新想法/坑点先追加到 `INBOX.md`
2. **会话结束**：从 INBOX 收敛到当天日志
3. **关键概念**：创建/更新 `mastery/<concept>.md`
4. **每周日**：执行周复盘

## 掌握状态标记

- ✅ 掌握：能复述 + 通过自测 + 能举例
- 🔄 进行中：能复述但不稳/不会举例/自测不过
- ⏳ 待开始：未开始学习
